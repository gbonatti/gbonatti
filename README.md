## 🚀 Projeto em Destaque: Model Risk Validation Suite

Neste projeto, estruturei uma esteira completa de validação técnica, simulando o dia a dia de uma área de Risco de Modelos:

* **Validação Independente:** Scripts para teste de performance e aderência de modelos.
* **Monitoramento:** Implementação de rotinas de backtesting e acompanhamento de data drift.
* **Stress Testing:** Simulação de cenários para riscos de liquidez e taxas de juros (IRRBB).
* **Prevenção:** Algoritmos para detecção de anomalias em dados transacionais.
